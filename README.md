# vcn-worldcup-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，不针对任何单个域名或具体网站，仅作为网络内容存档用途。

## 目录说明

```
.
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面引用的资源文件（如图片、样式、脚本）
├── index.html      # 页面索引或入口文件（可选）
└── README.md       # 本文件
```

- `pages/`：所有归档的 HTML 页面存放于此，按日期或主题子目录组织。
- `assets/`：与页面相关的静态资源，确保页面可离线浏览。
- `index.html`：可选的导航页面，用于展示归档列表。

## 页面归档说明

- 所有页面均为完整的独立 HTML 文件，不依赖外部服务器或数据库。
- 归档内容可能包含但不限于：信息展示页面、活动记录页面、数据可视化页面等。
- 页面内容来源于公开可访问的网络资源，仅作存档与展示用途。
- 不保证所有页面中的链接、脚本或外部资源长期有效。

## 维护说明

- 本仓库由维护者不定期更新，新增或移除归档页面。
- 欢迎通过 Issue 或 Pull Request 提交页面归档请求或修正建议。
- 提交归档页面时，请确保：
  - 页面内容不违反相关法律法规。
  - 页面不包含恶意代码或跟踪脚本。
  - 页面可独立运行，无需额外服务端支持。
- 维护者保留对归档内容进行筛选、编辑或删除的权利。
